# python-template
This repository is a template for python projects


### Arquitetura do Projeto:

~~~
Project
│   .gitignore
│   docker-compose.yml
│   Dockerfile
│   LICENSE
│   main.py
│   Makefile
│   README.md
│   requirements.txt
│
├───infrastructure
│       __init__.py
│
├───settings
│       api.env
│
└───src
    │   __init__.py
    │
    ├───api
    │       __init__.py
    │
    ├───common
    │   │   __init__.py
    │   │
    │   ├───config
    │   │       __init__.py
    │   │
    │   └───models
    │           __init__.py
    │
    ├───test
    │       __init__.py
    │
    └───utils
            __init__.py
~~~
